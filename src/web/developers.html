There are a number of API calls available within monroeminutes.org - all of which are free to use.

Search API

http://monroeminutes.org/search.json
Required Fields:
phrase - the phrase you are searching for within the index.
Optional Fields:
orgid - the ID of the organization you are interested in. See the getorgs API documentation for more info on OrgID.
bodyid - the ID of the body you are interested in. See the getorgs API documentation for more info on BodyID.
Notes:
orgid or may be spcified, not both
Example:
Search the entire database for 'welcome' - http://monroeminutes.org/search.json?phrase=welcome
Search only Body ID 2 for 'fracking' - http://monroeminutes.org/search.json?phrase=fracking&bodyid=2
